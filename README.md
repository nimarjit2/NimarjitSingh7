💻 Simple Interest Calculator using Python

As part of strengthening my programming fundamentals, I built a Simple Interest Calculator using Python in Google Colab.

The program takes user inputs such as Principal Amount, Rate of Interest, and Time Period, and calculates both the Simple Interest and the Total Amount using Python.

🔎 Key concepts used:

Taking user input in Python

Performing financial calculations using formulas

Displaying computed results clearly

Running Python code in a cloud environment using Google Colab

📊 Formula used:

Simple Interest = (Principal × Rate × Time) / 100

The program then calculates the total amount payable by adding the interest to the principal.

💡 Key takeaway:
Building small financial calculators like this helps strengthen Python fundamentals, logical thinking, and basic financial computation skills.
